# Demo Guide

## Run

```bash
docker compose up --build
```

Open `http://localhost:5173`.

## Login Credentials

- Admin: `admin@sau.ac.in / admin123`
- Warden: `warden@sau.ac.in / warden123`
- Student: `student@sau.ac.in / student123`

## 30-Second Walkthrough

1. Sign in as `admin@sau.ac.in`.
2. Show the dashboard stats, recent complaints, and announcements panel.
3. Open `Students` and approve or review a student profile with roommate suggestions.
4. Open `Maintenance` and show task submission plus status updates.
5. Switch to the student account and show complaint tracking, maintenance requests, and announcements.
